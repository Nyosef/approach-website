# Approach App Landing Page

A professional single-page website designed to attract dating coaches and explain the value proposition of the Approach App platform.

## Features

- **Modern Design**: Dark blue gradient background with colorful accent cards
- **Compelling Statistics**: Cold approach vs dating app success rates
- **Coach-Focused Content**: Revenue opportunities and client engagement benefits
- **App Features Showcase**: Support Hub, Training, Communication, Journaling, Daily Inspirations
- **Contact Integration**: Direct links to email, Instagram, and LinkedIn
- **Mobile Responsive**: Optimized for all devices
- **Fast Loading**: Single HTML file with embedded CSS and minimal JavaScript

## Quick Start

Simply open `index.html` in any web browser to view the landing page.

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Upload the `index.html` file to the repository
3. Go to repository Settings > Pages
4. Select "Deploy from a branch" and choose "main"
5. Your site will be available at `https://yourusername.github.io/repositoryname`
